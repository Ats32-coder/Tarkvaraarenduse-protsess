While-tsükkel Pythonis

While-tsükkel erineb for-tsüklist selle poolest, et selle käivitamise arv ei ole rangelt määratud. Tsüklile antakse boolean-väide, mida kontrollitakse iga iteratsiooni alguses. While-tsükkel kestab, kuni väide on tõene.

While-tsükli süntaks


while boolean_statement:
                # kood, mis täidetakse, kuni väide on True

Näide: järjendi väärtuste läbikäik


i = 0
            list = [3, 2, 10]
            while i != len(list):
                print(list[i])
                i += 1

Konsooli väljund:

3
            2
            10

Lõpmatu tsükkel

while True:
                # kood, mis käib lõpmatuseni
                pass  # pass on tühikohatäitja

While-tsükkel juhuslike väärtuste töötlemisel


While on eriti kasulik, kui vajame tsüklit määramatu arv kordi, näiteks juhuslike väärtuste või kasutaja sisendiga töötamisel. 

import random as r

            check = 10            # number, mida soovime saada
            random_choose = 0

            while random_choose != check:  # seni, kuni ei saa õiget numbrit
                random_choose = r.choice(range(11))  # juhuslik number 0-10
                print(random_choose)

Sel juhul on tsüklite arv ennustamatu ja erineb iga programmi käivitamisel.
