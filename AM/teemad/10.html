For-tsükkel Pythonis

For-tsüklit kasutatakse siis, kui on vaja jada (Pythonis näiteks sõne, järjend või ennik) kõik elemendid läbi käia ning iga elemendiga midagi teha.

Lihtne näide sõne puhul


nums = "12345"  # sõne, mida tsükliga käiakse läbi
            for i in nums:       # for-tsükli definitsioon
                print(i)         # kood, mis täidetakse iga elemendi kohta

Konsooli väljund:

1
            2
            3
            4
            5

Tsüklimuutuja nimetus


Tsüklimuutuja (control variable) võib olla mis tahes nimi, mis kirjeldab elemente, mida tsükliga käiakse läbi. Näiteks sõne puhul võiks nimeks olla letter: 

alphabet = "abcdefghijklmnopqrstuvwxyz"
            for letter in alphabet:
                print(letter)

Järjendi elementide läbikäik:

cats = ["Whiskers", "Mittens", "Luna", "Oliver"]
            for cat in cats:
                print(cat)

Väljund:

Whiskers
            Mittens
            Luna
            Oliver

For-tsükli sees return kasutamine


Kui soovime tsükli varem lõpetada ja tagastada väärtuse, kasutatakse return. Näide sõne numbrite eemaldamiseks: 

def remove_numbers(string_with_numbers: str):
                numbers_removed = ""  # uus sõne ainult tähtedega
                for character in string_with_numbers:
                    if character.isalpha():   # kui täht
                        numbers_removed += character
                return numbers_removed       # tagastatakse alles pärast tsükli lõppu

            print(remove_numbers("sl42dap1"))  # tulemuseks "sldap"

Funktsiooni valideerimise näide: tagastame False kohe, kui sõne sisaldab numbreid: 

def validate_string(string_to_validate: str):
                for character in string_to_validate:
                    if not character.isalpha():  # kui ei ole täht
                        return False             # kohe katkestame tsükli
                return True                      # kui kõik tähed, tagastame True

            print(validate_string("sl42dap1"))  # tulemuseks False
