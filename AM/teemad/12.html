Sõne (string) Pythonis

Sõne ehk string on andmetüüp Pythonis, mida kasutatakse teksti esitamiseks. Sõne kirja panemiseks kasutatakse ülakomasid (" ") või jutumärke (' '). Mitmerealine sõne tuleb piiritleda kolmekordsete jutumärkidega (""" """).

Pythonis on sõne tüübi tähistuseks str.

Sõne loomine


# Ülakomade kasutamine
            str_quotes = "hello"

            # Jutumärkide kasutamine
            str_commas = 'ABC'

            # Kolmekordsed jutumärgid mitmerealise teksti jaoks
            str_triple = """
            _____         _    _
            |  __ \       | |  | |
            | |__) |_   _ | |_ | |__    ___   _ ___
            |  ___/| | | || __|| '_ \  / _ \ | '_' |
            | |    | |_| || |_ | | | || (_) || | | |
            |_|     \__, | \__||_| |_| \___/ |_| |_|
                    __/ |
                    |___/
            """

Kombinatsioonid ja erimärgid


Kui tekstis on ülakomasid, tasub sõne piiritleda jutumärkidega ja vastupidi. Langkriips (\) on spetsiaalne sümbol, mille kasutamine ülakoma või jutumärgi ees annab Pythonile teada, et tegemist pole sõne lõpuga. 

# Näide erilise sümboli kasutamisest
            quote_in_string = "See on ülakoma: \" "
            apostrophe_in_string = 'See on apostroof: \' '
